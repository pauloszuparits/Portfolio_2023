Sobre o projeto

Este projeto teve como objetivo fazer um gerenciador para uma locadora de filmes, utilizando do processamento de um servidor através de sockets. O software possui diversas fuincionalidades que auxiliam no gerenciamento de uma locadora. O usuário pode cadastrar, altera e remover filmes e clientes, e também, alugar um filme que será vinculado a um cliente, entre outras funcionalidades.Saiba mais sobre este projeto, acesse meu GitHub
